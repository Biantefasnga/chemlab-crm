# ChemLab Sales CRM v2.0

CRM untuk PT Integrasi Lab Indonesia — dibangun dengan React + Firebase.

## Fitur
- Login dengan akun Google (Firebase Authentication)
- Database cloud real-time (Firebase Firestore)
- Dashboard KPI, Ongoing Services, Service Tracker
- Manajemen Klien & Leads, Service Catalog
- Kalkulasi Insentif Otomatis, Manajemen Sales Karyawan

## Deploy ke Vercel
1. Push ke GitHub
2. Connect repo di vercel.com
3. Deploy — selesai!
